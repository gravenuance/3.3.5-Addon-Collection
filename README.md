## Addon bundle

Ace3 is required for many of these addons.

Most have been **re-written**, some are new, and several have gained significant new features. [web:17]

---

### Included addons

- **!!!Zoom!!!**  
  Re-written to handle garbage collection more cleanly.

- **BlizzMove**  
  Re-written to avoid random bugs and improve overall stability.

- **BlueRedRange**  
  New addon that functions similarly to `TullaRange`.

- **CombatLogFix**  
  Re-written and now includes an additional combat log fix.

- **CompactNameplates**  
  Re-written with many new configuration options and features.

- **Doom_CooldownPulse**  
  New behavior: disables the button when the effect is not active.

- **OmniCC**  
  Includes fixes for long-standing issues, such as timers not resetting correctly on `ZONE_CHANGE`.

- **OneBag3**  
  Re-written for better performance and maintainability.

- **RetailUI**  
  Re-written and cleaned up, with multiple bugs fixed.

- **SimpleCombatLogger**  
  Backported for compatibility with this client.

- **SnowfallKeyPress**  
  Re-written and slightly adjusted to feel more responsive.
